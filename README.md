# AI CV Analyzer

AI-powered web app that analyzes your CV against any job description.

## Features
- Upload PDF CV or paste text
- Match score out of 10
- Matched and missing skills
- 3 improvement suggestions
- Analysis history saved

## Tech Stack
Python, Flask, OpenAI GPT-4o-mini, SQLite, Bootstrap 5
